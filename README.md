# 🚦 GhanaFlow — AI Traffic Intelligence Platform

> **TechHub Africa Hackathon 2026 | Track: Traffic Mitigation**

GhanaFlow is a real-time, AI-powered traffic intelligence platform designed to solve Ghana's urban congestion crisis. It combines citizen crowdsourcing, artificial intelligence, and live mapping to give drivers, city planners, and emergency services a unified view of Accra's roads.

---

## 🎯 Problem

Accra loses millions of hours annually to traffic congestion. There is no centralized, real-time system that:
- Collects live incident reports from citizens
- Analyzes conditions using AI
- Suggests optimal routes instantly
- Gives the government actionable data

GhanaFlow solves all four.

---

## 💡 Solution

A three-layer intelligence platform:

| Layer | What it does |
|-------|-------------|
| 🗺️ **Live Map** | Real-time incident markers across Accra's key corridors |
| 👥 **Citizen Reports** | Anyone can report accidents, flooding, or heavy traffic |
| 🤖 **AI Analysis** | Every report is instantly analyzed — severity, impact, alternate routes |

---

## ✨ Key Features

- **Real-time Traffic Map** — Interactive map of Accra with color-coded congestion zones
- **Citizen Incident Reporting** — Submit text, location, or photo reports instantly
- **AI-Powered Analysis** — Automatic classification of incident type, severity & response
- **Smart Route Suggestions** — AI recommends fastest alternate routes in real time
- **Government Dashboard** — Live statistics: active incidents, average speed, congestion %
- **Hourly Heatmap** — Visual congestion patterns by day and hour
- **Live Incident Feed** — Chronological feed of all active incidents with AI notes

## 🚀 How to Run

No installation required. Just open the file:

```bash
# Option 1 — Direct browser
Open index.html in any modern browser (Chrome, Firefox, Edge)

# Option 2 — Local server
npx serve .
# Then visit http://localhost:3000
```

---

## 📱 How It Works

1. **Citizens** spot an incident (accident, flooding, heavy traffic)
2. They open GhanaFlow and submit a report with location + description
3. **AI instantly analyzes** the report — classifies severity, estimates delay, suggests detours
4. The **live map updates** for all users in real time
5. **Government dashboard** tracks patterns and deploys resources accordingly

---

## 🏆 Impact

- Reduces average commute time by providing real-time alternate routes
- Empowers citizens to actively contribute to city infrastructure
- Gives Ghana's traffic authorities data-driven insights for the first time
- Scalable to all major Ghanaian cities: Kumasi, Tamale, Takoradi
